# 👶 Child Daily Routine App

A simple and beginner-friendly web application to help children follow a healthy daily routine with proper food, sleep, and prayer habits.

---

## 🚀 Features

- 🍽 Daily Food Plan (Indian meals with recipes)
- 😴 Sleep Schedule (bedtime & nap time)
- 🙏 Daily Prayers (English & Hindi)
- 🔄 Update Sleep Time from UI
- ➕ Add new meals dynamically

---

## 🛠 Tech Stack

- Frontend: HTML, CSS, JavaScript  
- Backend: Python Flask  
- Data: Python (JSON-like structure)
